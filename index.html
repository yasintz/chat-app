# Development

You can start dbdocs server with pnpm nx serve-script hasura. it will push your changes automatically to dbdocs.io/yasinntazeoglu/ChatAppDevopen in new window. You can see new changes after page refresh. You don't need to deploy it to prod app manually. Github CI will deploy it when you push your changes to master branch in github.
